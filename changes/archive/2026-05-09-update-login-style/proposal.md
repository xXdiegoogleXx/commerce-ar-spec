## Why

The current login page needs a modern visual update to match the branding guidelines and provide a more polished user experience. The existing login style lacks visual refinement and doesn't reflect the product's premium positioning.

## What Changes

- Update color palette to use Lumiere brand colors (primary green #4e6700, lime accent #c0f429)
- Implement glassmorphism panel effect with backdrop blur
- Update typography to use Manrope (headlines) and Inter (body) font families
- Redesign input fields with minimal bottom border style
- Add Material Symbols icon library for icons
- Improve spacing and layout for better visual hierarchy
- Update button styling with hover effects and transitions

## Capabilities

### New Capabilities
- `login-redesign`: New login UI implementation with glassmorphism effects and modern typography

### Modified Capabilities
- `login-ui`: Update existing login UI requirements with new visual specifications

## Impact

- Frontend: Login page component updates in React/Vite project
- Theme: Tailwind CSS configuration updates
- Fonts: Add Manrope and Inter font families
- Icons: Add Material Symbols Outlined library
