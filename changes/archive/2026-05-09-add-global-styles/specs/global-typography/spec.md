## ADDED Requirements

### Requirement: Headline typography
El sistema DEBE configurar la fuente "Manrope" para uso en headlines y títulos principales.

#### Scenario: Headline font family
- **WHEN** se usa la variable `--dc-font-headline`
- **THEN** se aplica la familia de fuente "Manrope"

### Requirement: Body typography
El sistema DEBE configurar la fuente "Inter" para uso en texto de cuerpo.

#### Scenario: Body font family
- **WHEN** se usa la variable `--dc-font-body`
- **THEN** se aplica la familia de fuente "Inter"

### Requirement: Label typography
El sistema DEBE configurar la fuente "Inter" para uso en etiquetas y texto pequeño.

#### Scenario: Label font family
- **WHEN** se usa la variable `--dc-font-label`
- **THEN** se aplica la familia de fuente "Inter"
